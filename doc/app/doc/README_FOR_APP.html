Architecture

This file is for development doc, /README is for users documentation. please update CHANGELOG to state any new functionality or amend on existing functionality.

All files named application are meant for things that applies and make sense application wide. Use then application_layout to refer to anything that should apply only to the layout and not to the templete, despite things written in there may apply to templates as well. E.g.: br_tag is defined in application_helper.rb because it is meant to be used all along the app, au contraire flash_messages belongs to application_layout because despite the method is available in templates as well, it makes sense only in the layout.

Stylesheets

In my understanding all the style files must be in .css.scss format because application.css.scss loads and compile them all together.
